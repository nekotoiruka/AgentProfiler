Agent Profiler
GitHub
AI-Powered Agent Creation

あなたの分身を
つくる。動かす。観察する。

47の質問であなたの思考特性・価値観・意思決定パターンを解析。パーソナライズされたAIエージェントを生成し、他者の分身との対話から新たな自分を発見する。

GitHub で見る → 機能を見る
◆

4軸思考特性プロファイリング

独自の4軸モデルで思考傾向をスコアリング。LLM による自由記述分析と組み合わせ、高精度な性格プロファイルを生成。

◎

ペルソナレジストリ

分身を「公開」すると共有プールに登録。他ユーザーの分身とチャット・議論が可能に。異なる価値観同士のぶつかり合いから新たな視点を発見。

△

マルチエージェント議論

2〜6体の分身がテーマについて自律的に議論。異なる視点からの洞察をリアルタイムで観察。

💡

インサイト抽出

議論完了後、AI が対話を分析。主要な気づき・対立点・予想外の視点・具体的なアクション提案を人間にフィードバック。

○

Agent Pack Zip エクスポート

VSCode / GitHub Copilot / Claude Code 向けの構成資産を自動生成。解凍して配置するだけで分身が動く。

●

相性診断 & レコメンド

4軸ベクトルの Cosine Similarity で分身間の相性を診断。議論パートナーを自動マッチング。

使い方

1

質問に答える

47問の質問に回答。4択 + 自由記述であなたの思考パターンを多角的に解析。

2

分身を公開する

生成された AI 分身を確認し、「公開」を承認するとレジストリに登録。他者の分身と対話可能に。

3

議論させる

自分の分身と他者の分身を集めてディスカッション。AI が対話を分析し、新たな気づきをフィードバック。
